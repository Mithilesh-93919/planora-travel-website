# Planora-Travel-Website
# Overview

Planora is a smart travel planning website designed to help users organize and explore trips efficiently. The platform provides destination exploration, itinerary planning, travel recommendations, hotel and flight suggestions, and local food recommendations through an interactive and user-friendly interface.

The project focuses on delivering a clean travel planning experience using frontend web technologies and responsive UI design.

# Features
* Destination browsing and travel exploration
* Personalized trip planning interface
* Interactive image sliders
* Hotel and flight suggestion sections
* Local food recommendations for destinations
* Responsive navigation and multi-page layout
* User-friendly UI/UX design

# Technologies Used
* HTML
* CSS
* JavaScript
* VS Code
